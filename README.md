# tactical-button-mouse--HackPad-project
 A custom 16-button, dual-encoder wired mouse and macropad powered by a Seeed XIAO RP2040. Features an OLED dashboard for real-time cursor speed tracking and granular pixel-control for precision navigation.

  **Hardware Mapping:**
*   **Movement:** 4 buttons (Up, Down, Left, Right)
*   **Clicks:** Single Left, Double Left, Right Click, Text Select Toggle
*   **Macros:** 8 customizable workflow buttons
*   **Encoders:** 1x Vertical Scroll, 1x Cursor Speed Adjustment
*   **Display:** 0.91" OLED (Cursor Speed Dashboard)
